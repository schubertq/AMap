# AMap
Ionic2+AngularJS2 高德地图的基本使用
===
* 功能如下:
    * 1.自动搜索补全功能
    * 2.搜索周边和指定地名
    * 3.使用inappbrowser在app内部展示高德地点详情页

* 依赖插件:
    * $ ionic plugin add cordova-plugin-geolocation
    * $ ionic plugin add cordova-plugin-inappbrowser
